This site is designed to be honest about where its claims come from.

Wix remains the source of truth for platform behavior. Astro and Starlight remain the source of truth for
the documentation foundation used to build this site.

## How the AI beta should use these

The Ask AI beta should follow three rules:

1. Prefer local handbook content when the question is already answered clearly here.
2. Prefer official Wix references when the question concerns platform behavior, commands, or edge cases.
3. Refuse or narrow the answer when it cannot support the response with citations.

That keeps the site useful without pretending to outrank the people building the platform.
