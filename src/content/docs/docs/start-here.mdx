## What the Wix CLI actually is

The Wix CLI is a command-line workflow for creating, developing, previewing, and releasing Wix apps and Wix Headless projects.

The key mindset shift:

> Think **workflow**, not scaffolding.

It's not just a project generator. It's the system that carries you through the entire lifecycle -- from creating a project to registering extensions with Wix.

## What this handbook covers

This handbook is app-first. It focuses on the areas where people get stuck most:

- Dashboard pages and modals
- Site widgets and embedded scripts
- Backend APIs and secure logic
- Release behavior and extension registration
- Environment variables, permissions, and self-hosting

Headless is part of the CLI family, but it's not the best first learning path unless you already know your frontend lives outside Wix.

## What this is not

This project is not a replacement for the official Wix docs.

Wix changes. Product teams rename things. Commands evolve. The official docs stay canonical. This handbook exists to make them easier to understand and navigate.

:::tip[Which Wix CLI are you using?]
There are three products that share the name "Wix CLI." This handbook covers the **current unified Wix CLI** -- not the older CLI for Apps or CLI for Sites. If you're unsure which one your project uses, [check here](https://dev.wix.com/docs/wix-cli/guides/development/determine-which-cli-your-project-uses) before going further.
:::

## Check yourself

After the next few pages, you should be able to answer these confidently:

<Steps>

1. **Which surface does this feature belong to?** Dashboard, Site, or Backend -- every feature has a home.

2. **Am I using the right Wix CLI?** Current unified CLI, older CLI for Apps, or CLI for Sites -- they're different products.

3. **What are the 7 commands in the normal loop?** Login, dev, generate, build, preview, release, whoami.

4. **Why would preview differ from release?** Preview hosts a build. Release also handles platform registration.

</Steps>

If you can answer those, the rest of the platform becomes much easier.
