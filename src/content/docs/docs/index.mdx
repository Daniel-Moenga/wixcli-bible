## What Makes This Different

This handbook is built for the **current unified Wix CLI**, not a blend of old and new docs. It keeps Wix as the source of truth and says that clearly. It rewrites the subject for human learning instead of mirroring product docs verbatim. And it is Open-Source so the community can correct, extend, and improve it.

We also clarify things the official docs leave ambiguous -- like the difference between the three products all called "Wix CLI," why `preview` isn't the same as `release`, and the `extension.ts` vs `extensions.ts` naming inconsistency in the official project structure docs.
