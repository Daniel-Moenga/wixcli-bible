## What You'll Learn

This handbook covers the things that trip people up most:

- **Where your code runs** -- Dashboard, Site, or Backend. Pick wrong and everything feels broken.
- **The 7 commands that do 90% of the work** -- `login`, `dev`, `generate`, `build`, `preview`, `release`, and `whoami`.
- **Why preview and release are different** -- Preview hosts a build. Release registers extensions with Wix.
- **How to stop fighting the Wix Design System** -- When to use it, when you have freedom, and what breaks when you override it.
- **Project structure that actually makes sense** -- Which files you edit vs which files are generated machinery.
