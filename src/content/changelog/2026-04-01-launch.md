Wix CLI Bible launches as an open-source learning layer for the current Wix CLI.

### Included in v1

- a world-class landing page
- a structured docs handbook
- practical guides for common questions
- downloadable PDF archive
- open-source contribution and governance files

### Why this matters

The goal is not to compete with Wix documentation as the source of truth. The goal is to make the current
CLI easier to learn, easier to reference, and easier for the community to improve together.
