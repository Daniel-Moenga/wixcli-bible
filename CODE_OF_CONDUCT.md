# Code of Conduct

## Our standard

Wix CLI Bible is meant to be a welcoming learning project. We expect contributors to be respectful,
clear, and generous with context.

Examples of behavior that support a healthy project:

- helping beginners without condescension
- giving technical feedback without hostility
- crediting upstream sources honestly
- prioritizing accuracy over ego

Examples of unacceptable behavior:

- harassment or personal attacks
- dismissive or insulting responses to beginner questions
- deliberate misinformation
- hostile review behavior

## Enforcement

Project maintainers may remove or reject contributions that violate this standard.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and related community
spaces tied to the project.
